This project uses libraries to do various things.
The MovieDB API is used, in conjunction with the movieDB js library to get the images and data.
jQuery is used for logic.
Reset is used for easy cross-browser compatibility.
jQuery Transit is used (but not very much) for css-accelerated transitions but was not as useful as simply using animations as transition cannot easily be stopped.
Masonry was used for block arranging and re-arranging.
Finally, ReModal was used for modal dialogs.

Also, the TV icon used is from The Noun Project.
